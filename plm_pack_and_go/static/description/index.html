Plm Pack and Go

Odoo version covered : 11.0

This module allows you to download a zip file containing all documents attached to a BOM.

How it works:

  • Go to Search on Engineering Parts and select a product


  • Go to the action Menu




  • Select component to compute it's BOM
  • Select export type to define what types of files you want to export
  • Click the Compute Bom button to compute the BOM




  • All files and documents related to the BOM are splitted in 2D, 3D, PDF and other types in the nootebook. This allows you to remove unuseful files or add others.


  • In 2D and 3D tabs if available you can define the export type in the line.




  • To force all types in one time you can select your preferred convetion type in the Force Types selection field. Than press the Force Types button to change the value in all lines




  • To clear all documents of a type, for example 2D, you can click Clear button


  • When you are ready with the changes you can click Create zip archive button. The procedure will convert the files if needed and the zip will be created and will be available for download in the same wizard




Once you have the zip file in your machine .. unzip it and use it wherever you want.
